# 武昌首义学院 ACM 协会

## 💡 关于我们

武昌首义学院 ACM 协会由一群**热爱算法、喜欢编程、享受解决问题乐趣**的同学组成。我们专注于算法竞赛、程序设计能力提升，以及计算机科学的深入探索。无论你是刚接触编程的新手，还是渴望在算法竞赛中大展身手的选手，这里都能找到属于你的舞台。

## 🎯 协会宗旨

* 提升同学们的算法与编程能力
* 推动校园技术氛围建设
* 培养更多优秀的 ACM/ICPC、CCPC 程序设计选手
* 为有志于计算机方向的同学提供学习、交流与成长的环境

## 🚀 我们在做什么

* **算法课程与训练营**：从零基础语法到进阶算法，为不同水平的同学准备对应内容
* **每周例题与讲解**：精选高质量竞赛题目，带你理解背后的思维方式
* **组队参与竞赛**：包括 ICPC、CCPC、蓝桥杯、省赛等
* **技术分享会**：面向全校同学开放，内容包括算法、计算机基础、工程技术等
* **学习小组**：和志同道合的伙伴一起进步

## 🤝 我们欢迎谁

* 想学习编程但不知道如何开始的同学
* 想提升算法能力、准备各种比赛的同学
* 对技术、计算机科学充满好奇心的同学
* 有志于进入大厂研发岗、算法岗的同学

## 🧭 如何加入我们

欢迎加入武昌首义学院 ACM 协会，一起探索更大的世界！

可以通过以下方式联系我们：

* 现场报名：每学期初的社团招新现场
